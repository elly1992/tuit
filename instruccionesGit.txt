Tenemos que crear una estructura de carpetas:
C: -> crear carpeta go -> crear carptas src, pkg y bin.
Dentro de src crear la carpeta github.com y dentro de esta carpeta hay que crear otra con el nombre de usuario en github (elly1992), luego crear una carpeta con el nombre del repositorio a usar y dentro de esta carpeta crear las siguientes, bd, handlers, jwt, middlew, routers, models y uploads.
